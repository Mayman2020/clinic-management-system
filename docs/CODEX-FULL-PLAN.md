# خطة تنفيذ Codex المقترحة

| المرحلة | المدة التقريبية | الأهمية |
|---------|-----------------|---------|
| **Phase 0** | 2–4 أيام | إصلاحات فورية — deep links، صلاحيات، profile، SSE |
